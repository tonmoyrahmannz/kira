## Setup Instructions (Outlook on the Web)

These templates can be set up using Outlook's built-in **My Templates** add-in:

### Step-by-step

1. Go to **outlook.office.com** and sign in as tonmoy@whiroanalytics.co.nz
2. Click **New message** to open a compose window
3. In the compose window, click the **ellipsis (...)** (More actions) in the bottom toolbar
4. Select **My Templates** from the menu
   - If you don't see it, click **Get Add-ins** > search for "My Templates" > Add it
5. A pane opens on the right side. Click **+ Add template**
6. Enter a **Title** (e.g. "First Enquiry Response") and paste the template text in the body
7. Click **Save**
8. Repeat for all 8 templates

### How to use

1. Click **New message**
2. Click **...** > **My Templates**
3. Click on the template you want. It inserts into the email body.
4. Fill in [Name], [Org], and other placeholders
5. Swap the sender to contact@ if replying to a public enquiry
6. Send

### Alternative (if My Templates isn't available)

1. Create a folder in your mailbox called **Templates**
2. Save each template as a draft in that folder
3. When needed: open the draft > **Copy** > **Discard** > Paste into a new message

### Note on placeholder variables

The templates use `[Name]`, `[Org]` etc. These aren't automated merge fields. You'll need to manually replace them each time. Think of them as a fill-in-the-blank starting point, not mail merge.
