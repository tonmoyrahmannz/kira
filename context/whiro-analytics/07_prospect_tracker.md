# Whiro Analytics — Prospect & Opportunity Tracker

*Purpose: Lightweight CRM. One markdown file. Update after each conversation.*

---

## How to Use

- Add a new entry at the top when a prospect is identified
- Update status as the deal progresses
- Archive closed-won or closed-lost entries at the bottom
- Kira maintains this file and prompts Tonmoy for updates after notable interactions

## Monthly Review Prompts

- [ ] Any prospects stalled for >4 weeks? Follow up or move to archive?
- [ ] Any referral partners with pending commissions?
- [ ] Any leads from recent conversations not yet captured?
- [ ] Any services that need pricing or scope adjustment based on feedback?
- [ ] Capacity check — do you have time to take on new work this month?

## Notes for Kira

- Keep this file concise — one table per section, updated in place.
- After a Telegram conversation where Tonmoy mentions a new contact/lead, offer to add it here.
- If a prospect has gone silent for 4+ weeks, flag for follow-up or archive.
- This is not a CRM export — keep it human-readable and low-friction.
- When a deal reaches "Proposal Sent," create a separate note in `Agent_Notes/` with the proposal details.
